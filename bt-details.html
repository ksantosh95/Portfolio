Bug Tracker


Bug Tracking web application to aid software product development

Project Overview

Web application for efficiently tracking issues in software development lifecycle. Key features include Auth0 authorization, notification system, summary dashboard and comment functionality.
  • Technologies Used : Python, Flask, PostgreSQL, SQLAlchemy, Rest + Ajax, HTML/CSS
  • Project Timeline : Nov '20 - Dec '20
  • Demo Link: bugTrac

Approach & Solution

Phase 1 : Creating the Requirements specifications document
  • Requirement clearly noted for all views
  • Designed mock dashboards
Phase 2 : Planning
  • Set deadlines for each view and functionality
Phase 3 : Database Design
  • Designed physical data model with normalization
  • Inserted sample data for testing purposes
**Phase 4 & 5 were repeated for each view
Phase 4 : Backend Build
  • Decided end point routes for each metric
  • Built controllers and APIs for each functionality
Phase 5 : Build Front End
  • Developed web pages using HTML/CSS
  • Used JavaScript AJAX requests for fetching data from end points
Phase 6 : Testing
