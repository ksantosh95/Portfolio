Sentiment Analysis


Web Tool to determine whether a user feedback is positive or negative

Project Overview

Amazon SageMaker based machine learning model supported with a simple user interface to determine whether the text feedback submitted by the user is positive or negative.
  • Technologies Used : NodeJS, HTML, JavaScript, Oracle DB
  • Project Timeline : Oct '19 - Dec '19

Approach & Solution

Phase 1 : Data Pre-Processing
  • Labelled training data is used for pre-processing. Since the user feedback is a text object, all words are converted to lowercase and stop words are removed.
  • A word dictionary of 5000 most frequent words is created and stored in a matrix.
  • Each user response is then compared with the word dictionary and each bit from the 5000 words is set to 1 or 0 based on its existence.
  • Thus, each text response is converted into an array of binary numbers of size 5000.
Phase 2 : Training & Testing the Model
  • XgBoost model is used for training.
  • Model is evaluated against the validation set.
Phase 3 : Deploying the model
  • Simple HTML UI is created for accepting a user response and initiating a HTTP request to an endpoint.
  • A Lambda function is set up on AWS to run whenever it receives a request.
  • An API Gateway is set up next to trigger the lambda function and access the endpoint
