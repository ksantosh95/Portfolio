Lua Compiler


Compiler written in Java for LUA programming language

Project Overview

This project consists of a scanner, parser and interpreter for Lua programming language. The grammar for Lua language is referred to compile the inputs. Program is written in Java with ASM byte code framework.
  • Technologies Used : Java, ASM framework
  • Project Timeline : Sept '19 - Dec '19

Approach & Solution

Phase 1 : Scanner.
  • Built a scanner process to read the input from a file and tokenize each word
Phase 2 : Parser
  • Classified each input token and grouped tokens of the same category
  • for instance "if x > y" would be read as 4 different tokens in step 1. These tokens were combined as an if statement in this phase.
Phase 3 : Interpreter
  • Each statement from phase 2 is evaluated and results are computed
