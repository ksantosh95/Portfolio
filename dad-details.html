Data Analytics Dashboard


Web Dashboard App to monitor the inspections carried at various Restaurants in Los Angeles City.

Project Overview

Developing a web application for tracking and analyzing the restaurant market dataset hosted by Los Angeles City. The dataset contains the inspection results for various restaurants and stores in LA city. Along with tracking the performance of each restaurant, the web application also captures trends in the restaurant behaviours thereby alerting their owners.
  • Technologies Used : NodeJS, HTML, JavaScript, Oracle DB
  • Project Timeline : Oct '19 - Dec '19

Approach & Solution

Phase 1 : Designing data model and importing data into databases.
  • Physical data model designed for tables with normalizaition
  • Tables created in Oracle database and data was imported.
Phase 2 : Designing mock dashboards & Designing test cases
  • Designed mock dashboards with all necessary metrics highlighted
  • Desinged test cases for boundary conditons.
Phase 3 : Configure API endpoints
  • Decided end point routes for each metric
  • Built controllers and APIs for each view
Phase 4 : Build Front End
  • Developed web pages using HTML/CSS
  • Used JavaScript AJAX requests for fetching data from end points
Phase 5 : Testing
